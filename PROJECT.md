# Setup Guide — Projekt-Steckbrief

## Beschreibung

Interaktiver Setup Guide fuer Multi-Agent Development mit Claude Code. Erklaert den Weg von einfacher AI-Chat-Nutzung (L1) bis hin zu autonomen Agent-Teams (L4) und zeigt Entwicklern Schritt fuer Schritt, wie sie GitHub, Cloud-Infrastruktur und Claude Code einrichten. Publiziert als GitHub Pages Website.

## Kontext

- **Tech Stack:** Jekyll (Static Site Generator) + Cayman Theme + GitHub Pages
- **Content-Format:** Markdown + inline HTML/CSS + SVG-Diagramme (Team-Organigramm, Orchestrator-Flow, Message-Flow)
- **URL:** https://sutermarkus-rakoon.github.io/setup-guide/
- **Aktuelle Phase:** v1.1 — Basic Tab (GitHub + Cloud Setup) und Advanced Tab (Multi-Session Classic + Autonomous) fertig. 5 Autonomie-Level dokumentiert (L1–L5).
- **Abhaengigkeiten:** GitHub Pages Hosting, Jekyll remote theme (cayman), keine Build-Dependencies

## Zielarchitektur

- **Struktur:** Tab-basierte Navigation (Overview / Basic / Advanced) mit Sub-Tabs im Advanced-Bereich (Classic / Autonomous)
- **Inhalte:** Vollstaendige Anleitung fuer alle 5 Autonomie-Level, von Chatbot (L1) bis Self-Improving Systems (L5). Praktische Quick-Start-Checklisten, ASCII-Diagramme, Vergleichstabellen.
- **Visuell:** SVG-Grafiken fuer Architektur, Team-Struktur und Workflow. Screenshots vom Agent Monitor Dashboard.
- **Qualitaetsziele:** Vollstaendig auf Englisch, visuell ansprechend mit Diagrammen statt nur Fliesstext, leicht verstaendlich fuer Einsteiger
- **Erweiterbarkeit:** Neue Tabs/Sektionen bei Bedarf ergaenzbar (z.B. Expert Agents, CI/CD Integration)
